## 🔮 Predictive Models

### Solver Optimization Framework

Implementation of a least-squares based optimization model to estimate team strength and home advantage across leagues.

📁 **Location:**  
`predictive-models/`

Includes:
- NFL Game Predictor  
- NBA Game Predictor  

**Key Techniques:**
- Strength rating modeling  
- Home advantage estimation  
- Optimization via least squares  
- Model validation
