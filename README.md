# Sports Analytics Portfolio

This repository showcases a collection of sports analytics projects spanning probability modeling, descriptive analysis, predictive modeling, and optimization techniques. The goal is to demonstrate how data can be used to analyze performance, predict outcomes, and support data-driven decision-making.

## 📂 Repository Structure

### 🎲 Probability Models
Located in: `Probability_Models/`

- Focuses on modeling randomness and streak behavior in sports performance  
- Explores probabilistic patterns and variability in outcomes  
- Example: streak analysis and probability-based modeling  

---

### 📈 Descriptive Analytics
Located in: `descriptive-analytics/`

- Performs exploratory data analysis using R on NBA player performance data  
- Computes key statistical measures including mean, median, standard deviation, variance, minimum, and maximum  
- Visualizes distributions using histograms and boxplots  

**Analysis Focus:**
- Points per Game (PTS)
- Minutes Played (MP)
- Total Rebounds (TRB)

**Techniques Used:**
- Distribution analysis (histograms)
- Outlier detection (boxplots)
- Summary statistics (five-number summary and central tendency)

**Tools:**
- R Programming
- Base R visualization functions

---

### 🤖 Predictive Models
Located in: `predictive-models/`

This section focuses on building models to predict outcomes and optimize decisions in sports analytics.

**Projects included:**

- 🏀 **NBA Game Predictor**  
  Predicts game outcomes using historical performance data  

- 🏈 **NFL Game Predictor**  
  Forecasts NFL results using statistical modeling techniques  

- 🎯 **Fantasy Team Optimization & Performance Prediction**  
  Combines regression modeling and optimization techniques to:
  - Predict player performance across seasons  
  - Select the optimal team under budget constraints using Excel Solver  
  - Demonstrate both predictive and prescriptive analytics
