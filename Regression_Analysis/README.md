### 📉 Regression Analysis
 
Located in: `regression-analysis/`
 
Applies simple and multiple linear regression to NFL and NBA data to identify statistical predictors of real-world sports outcomes.
 
**Projects included:**
 
* 🏈 **NFL Simple Linear Regression**
  Examines whether total yards gained per game predicts points scored, using R to build the model, generate a scatter plot, and interpret R-squared and p-value outputs.
* 🏀 **NBA Salary Prediction (Multiple Regression)**
  Builds two competing salary prediction models using player performance stats:
  + Model 1 (Offensive): Points, Assists, FG Made, FG%
  + Model 2 (Defensive): Blocks, Steals, Defensive Rebounds
  Compares which model better explains salary variance and reflects how the NBA market values different types of contributions.
**Techniques Used:**
 
* Simple linear regression
* Multiple linear regression
* Model comparison and interpretation (R-squared, p-values, coefficients)
* Scatter plot visualization with regression line
**Tools:**
 
* R / RStudio
